# Skill: Comunicação Interna (comunicacao-interna)

Esta skill deve ser usada sempre que for solicitado a redação de algum tipo de comunicação interna, como relatórios de status, atualizações para a liderança, relatórios de incidentes ou atualizações de projeto.

## Quando usar esta skill

Para redigir comunicações internas, use esta skill para:

-   **Relatórios 3P** (Progresso, Planos, Problemas)
-   **Atualizações de Projeto**
-   **Relatórios de Incidentes**
-   **Respostas a FAQs** / Artigos de Base de Conhecimento
-   **Atualizações para a Liderança**

## Como usar esta skill

Para redigir qualquer comunicação interna:

1.  **Identifique o tipo de comunicação** a partir da solicitação do usuário.
2.  **Carregue o arquivo de diretrizes apropriado** do diretório `examples/`:
    -   `examples/relatorio-3p.md` - Para atualizações de time no formato Progresso/Planos/Problemas.
    -   `examples/relatorio-incidente.md` - Para relatórios de pós-mortem de incidentes técnicos.
    -   `examples/faq-base-conhecimento.md` - Para responder a perguntas frequentes ou documentar um fluxo.
    -   `examples/comunicado-geral.md` - Para qualquer outra comunicação que não se encaixe explicitamente em um dos formatos acima (ex: atualizações para liderança, anúncios de release).
3.  **Siga as instruções específicas** no arquivo carregado para formatação, tom e coleta de conteúdo.

Se o tipo de comunicação não corresponder a nenhuma diretriz existente, use o `comunicado-geral.md` como base e peça esclarecimentos ou mais contexto sobre o formato desejado.

## Palavras-chave

Relatório 3P, atualização semanal, comunicado interno, FAQs, perguntas frequentes, atualização de projeto, relatório de status, pós-mortem, RCA (Root Cause Analysis).
